## How to use

**Clone repo Windows**

```shell
git clone https://github.com/lobanov4real/lazynvim $env:LOCALAPPDATA\nvim
```

**Clone repo Linux**

```shell
git clone https://github.com/lobanov4real/lazynvim $HOME/.config/nvim
```
